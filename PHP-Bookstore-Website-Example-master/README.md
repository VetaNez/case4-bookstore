## Описание
Web-приложение книжного магазина с интерфейсами администратора и пользователя.

## Функционал
- Просмотр каталога книг
- Сортировка по категориям, авторам, году
- Покупка и аренда книг
- Админ-панель: управление книгами, ценами, статусами
- Автоматические уведомления об аренде

## Технологии
- PHP
- MySQL
- HTML/CSS
