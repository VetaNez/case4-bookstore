## Описание проекта
Web-приложение книжного магазина с двумя интерфейсами: для администратора и пользователя.


### Пользовательский интерфейс
- Просмотр каталога книг
- Сортировка по категориям, авторам и году написания
- Покупка книги
- Аренда книги (на 2 недели / месяц / 3 месяца)

### Административный интерфейс
- Добавление и редактирование книг
- Управление ценами
- Изменение статусов доступности
- Автоматические уведомления об окончании срока аренды

## Технологии
- Next.js
- TypeScript
- Tailwind CSS
- Prisma / PostgreSQL
