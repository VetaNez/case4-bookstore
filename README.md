## Описание проекта
Web-приложение книжного магазина с двумя интерфейсами: для администратора и пользователя.


### Пользовательский интерфейс
- Просмотр каталога книг
- Сортировка по категориям, авторам и году написания
- Покупка книги
- Аренда книги (на 2 недели / месяц / 3 месяца)
